# Blackthorn Field Notes — v0.9.1

## Release Theme

Stability, readability, and trust.

This release focuses entirely on layout stabilization, spacing consistency, accessibility verification, responsive QA, and cleanup work before future feature expansion.

---

# Phase 1A — Mobile Layout & Spacing Stabilization

## Goals

- Normalize homepage spacing rhythm
- Fix Featured vs Latest section spacing collapse
- Improve mobile readability and hierarchy
- Verify responsive consistency across devices

## Planned Work

### Homepage Section Rhythm

Audit and normalize spacing between:

- hero
- search
- featured section
- latest section

### Featured vs Latest Layout Fix

Resolve the mobile spacing issue where the Latest Field Notes heading visually collapses into the Featured Field Notes section.

Expected structure:

```text
Featured Field Notes
[ content ]

(clear spacing)

Latest Field Notes
[ article feed ]
```

### Mobile Typography Rhythm

Verify:

- heading spacing
- paragraph spacing
- excerpt spacing
- tap spacing
- line-height consistency

### Responsive QA

Test:

- iPhone Safari
- Android Chrome
- tablet portrait
- tablet landscape
- desktop responsive layouts
