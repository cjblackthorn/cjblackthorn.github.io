# Blackthorn Field Notes — Homepage Article Hierarchy Refinement

## Purpose

This refinement pass focuses on improving homepage article readability, spacing rhythm, metadata hierarchy, and mobile scanning behavior.

The goal is making the homepage feel calmer, cleaner, and more publication-oriented without redesigning the site.

---

# Refinement Targets

## Article Title Hierarchy

Improve:

- spacing beneath article titles
- visual separation from metadata
- mobile readability rhythm
- scanability between articles

---

## Metadata Readability

Normalize:

- spacing beneath metadata
- opacity consistency
- visual hierarchy beneath titles
- mobile readability

Desired structure:

Title

Metadata

Excerpt

Clear separation

Next article

---

## Excerpt Rhythm

Improve:

- paragraph breathing room
- mobile readability
- visual separation between article previews
- spacing consistency across homepage lists

---

## Mobile Scanning Behavior

Verify:

- no cramped article stacking
- readable spacing at smaller widths
- easier visual scanning
- improved tap readability

Test:

- iPhone Safari
- Android Chrome
- tablet portrait
- tablet landscape

---

# Non-Goals

This refinement does not include:

- homepage redesigns
- card redesign systems
- animations
- branding changes
- JavaScript systems
- navigation restructuring

---

# Desired End State

The homepage should increasingly feel like a calm editorial publication rather than a generic blog template.

The writing should remain the visual focus.

---

# Release Classification

Enhancement / UX Refinement / Typography / Mobile Readability
